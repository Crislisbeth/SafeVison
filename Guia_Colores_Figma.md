# Guía de Colores de SafeVision para Figma

Este documento detalla exactamente los colores que se utilizan en cada componente y pantalla de la aplicación, para que puedas diseñar en Figma con los colores reales.

## 1. Pantalla de Login (Inicio de Sesión)

| Elemento                                | Color                     | HEX                        |
| :-------------------------------------- | :------------------------ | :------------------------- |
| **Fondo de la pantalla**          | Guinda Más Oscuro        | `#4A0F23`                |
| **Tarjeta Blanca (Fondo)**        | Blanco Puro               | `#FFFFFF`                |
| **Línea superior de la Tarjeta** | Degradado Guinda a Dorado | `#8C1D40` ➔ `#E8A825` |
| **Ícono / Logo (Fondo)**         | Guinda Principal          | `#8C1D40`                |
| **Cajas de texto (Email/Clave)**  | Gris Claro                | `#F0F2F5`                |
| **Botón de "Entrar"**            | Guinda Principal          | `#8C1D40`                |
| **Textos principales**            | Casi Negro                | `#1A1A2E`                |

---

## 2. Menú Lateral (Sidebar)

| Elemento                                 | Color                     | HEX                       |
| :--------------------------------------- | :------------------------ | :------------------------ |
| **Fondo completo del Menú**       | Guinda Principal          | `#8C1D40`               |
| **Ícono cuadrado superior**       | Dorado Principal          | `#E8A825`               |
| **Texto de los botones inactivos** | Gris Claro / Atenuado     | `#A0AAB8`               |
| **Botón Activo (Fondo)**          | Dorado Transparente (15%) | `rgba(232,168,37,0.15)` |
| **Botón Activo (Texto e Ícono)** | Dorado Principal          | `#E8A825`               |



---

## 3. Barra Superior (Header)

| Elemento                                     | Color                    | HEX                     |
| :------------------------------------------- | :----------------------- | :---------------------- |
| **Fondo de la Barra**                  | Blanco Puro              | `#FFFFFF`             |
| **Título principal de la página**    | Guinda Principal         | `#8C1D40`             |
| **Hora y Fecha (Texto)**               | Gris Medio               | `#5A6070`             |
| **Indicador "Sistema Activo" (Fondo)** | Verde Transparente (10%) | `rgba(46,125,50,0.1)` |
| **Indicador "Sistema Activo" (Texto)** | Verde Éxito             | `#2E7D32`             |

---

## 4. Tarjetas de Estadísticas (Dashboard)

| Elemento                                | Color                              | HEX                            |
| :-------------------------------------- | :--------------------------------- | :----------------------------- |
| **Fondo de la Tarjeta**           | Blanco Puro                        | `#FFFFFF`                    |
| **Línea superior de adorno**     | Depende (Guinda, Dorado, etc)      | `#8C1D40` / `#E8A825`      |
| **Texto descriptivo (Etiqueta)**  | Gris Medio                         | `#5A6070`                    |
| **Número Grande (Estadística)** | Mismo color que su línea superior | `#8C1D40`, `#E8A825`, etc. |

---

## 5. Tablas y Datos (Detecciones)

| Elemento                                | Color      | HEX         |
| :-------------------------------------- | :--------- | :---------- |
| **Fondo de la Cabecera de Tabla** | Gris Claro | `#F0F2F5` |
| **Líneas divisorias (Bordes)**   | Gris Borde | `#E2E5EA` |
| **Texto de las filas (Datos)**    | Gris Medio | `#5A6070` |

---

## 6. Chips de Estado (Alertas visuales)

| Estado de Detección                 | Color de Fondo (10% opacidad) | Color del Texto (100%) | HEX del Texto |
| :----------------------------------- | :---------------------------- | :--------------------- | :------------ |
| **Éxito (Usa mascarilla)**    | Verde Transparente            | Verde Éxito           | `#2E7D32`   |
| **Peligro (Sin mascarilla)**   | Rojo Transparente             | Rojo Error             | `#C62828`   |
| **Advertencia (Casco)**        | Naranja Transparente          | Naranja                | `#E65100`   |
| **Neutro (Persona detectada)** | Dorado Transparente           | Dorado                 | `#E8A825`   |
